# Eyespie

Eyespie is a React Native application built with Expo, featuring advanced camera integration and AI capabilities.

## Features

- **Camera Integration**: Seamlessly capture photos using `expo-camera`.
- **AI-Powered**: Utilizes `cactus-react-native` for on-device AI inference.
- **State Management**: Robust state management using `zustand`.
- **Dependency Injection**: Modular architecture with `tsyringe`.
- **Modern UI**: Built with `react-native-reanimated` and `expo-blur` for a polished user experience.

## Installation

To set up the project locally, follow these steps:

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd eyespie
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

## Running the App

Start the development server:

```bash
npm start
```

Run on Android:

```bash
npm run android
```

Run on iOS:

```bash
npm run ios
```

## APK File

The compiled APK file for Android is available in the root directory of this project:

- **[eyespie.apk](./eyespie.apk)**

You can install this directly on your Android device to test the application without setting up the development environment.
